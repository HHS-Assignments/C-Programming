# Opgave 2: Meerdere computers
a) Laat op je eigen PC (of lap PC) zien wat er moet gebeuren als je een server laat draaien
waarbij een client op een andere machine een request maakt naar deze server.
b) Moet er ook nog wat gebeuren op de machine van de client?
c) Kan je met telnet of curl op een Linux machine een request maken naar de server? En welke
data wordt er dan verstuurd?
d) Maak via je eigen client (C programma) een request naar een webserver op het internet. Je
maakt dus een simpel http request. Wat is de response van de webserver? En welk deel geeft
aan of het een correct request is en welke andere delen bevat de response? Een korte uitleg
van het http 1.1 protocol vind je oa hier:
https://developer.mozilla.org/en-US/docs/Web/HTTP/Overview#http_flo
